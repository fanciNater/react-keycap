## how to use chrome

#### 常用快捷键的使用

| 快捷键 | 说明 |
| --- | --- |
| ctrl+n | 打开新窗口 | 
| ctrl+shift+n | 打开新无痕窗口 |
| ctrl+t | 打开新的页面 |
| ctrl+tab | 切换下一个页面 |
| ctrl+shift+tab | 切换上一个页面 |
| ctrl+w | 关闭当前页面 |
| ctrl+shift+w | 关闭当前窗口 |
| alt+f | 打开浏览器菜单 |
| ctrl+shift+b | 书签栏切换 |
| ctrl+h | 查看浏览历史 |
| ctrl+j | 查看下载历史 |
| shift+esc | 打开浏览器任务管理器 |
| ctrl+f | 搜索浏览器  enter 下一个/ shift+enter 上一个 |
| ctrl+shift+delete | 删除浏览器缓存 |
| ctrl+shift+j | 打开控制台 |
| ctrl+shift+i | 打开浏览器检查器 |
| ctrl+shift+c | 选择网页元素 |
